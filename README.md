# GPT4Free CLI/TUI Client

Терминальный клиент для взаимодействия с большими языковыми моделями через библиотеку GPT4Free (g4f). Проект включает в себя одиночные взаимодействия через командную строку флагом -p (Typer), а также полноценный интерфейс реализованный на фреймворке python (Textual).


## Функционал (В разработке)

- **Потоковый вывод**: ответ выводится не сразу, а последовательно.
- **Работа в CLI и TUI режимах**: можно получить одиночный ответ через флаг -p либо же запустить графический интерфейс.
- **Сохранение сессии**: после выхода выбранная модель и провайдер остаются в конфиге.


## Требования

- Python 3.10 и выше.
- Утилита `uv` для управления зависимостями.


## Установка и запуск (В разработке)

1. Клонируйте репозиторий:
   ```bash
   git clone [https://github.com/ваш-аккаунт/gpt4free-tui-cli.git](https://github.com/ваш-аккаунт/gpt4free-tui-cli.git)
   cd gpt4free-tui-cli

2. Запустите проект (зависимости установятся автоматически через uv):
   ```bash 
   uv run src/cli.py

## Команды CLI

Введите для вывода подсказок:
```bash
uv run src/cli.py --help
```

___

Ввод комманд в cli осуществляется путем указания флагов.

|Флаг|Назначение|Расшифровка|
|-|-|-|
|-p|промпт для нейросети|--prompt|
|-m|выбранная модель|--model|
|-pr|выбранный провайдер|--provider|


___

- ```uv run src/cli.py main -p "Привет"``` - пример вывода без указания сторонних флагов
- ```uv run src/cli.py main -p "Привет" -m deepseek -pr WeWordle``` вывод с указанием флагов модели и провайдера
